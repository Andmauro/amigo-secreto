# 🎁 Amigo Secreto

Aplicación web que permite agregar nombres de participantes y realizar un sorteo aleatorio para elegir el amigo secreto.

##  Funcionalidades

- Agregar nombres de participantes
- Mostrar la lista de participantes
- Realizar el sorteo aleatorio
- Mostrar el amigo secreto seleccionado

##  Cómo usar

1. Escribe el nombre de un participante.
2. Presiona el botón **Agregar**.
3. Cuando todos estén agregados presiona **Sortear amigo**.

##  Tecnologías utilizadas

- HTML
- CSS
- JavaScript
